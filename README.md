# 🧮 Calculator

A simple and responsive Calculator web app built using **HTML, CSS, and JavaScript**.  
It performs basic arithmetic operations with a clean and user-friendly interface.

---

## 🚀 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🧼 Clear button  
- ⚡ Fast and responsive UI  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## 📂 Project Structure

calculator/
│
├── index.html
├── style.css
└── script.js


---

## ▶️ How to Use

1. Open the project in your browser  
2. Click on numbers and operators  
3. Press "=" to get the result  
4. Use "C" to clear the screen
